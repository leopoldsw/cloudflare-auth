# create-cloudflare-auth

Create-package entrypoint for Cloudflare Auth starter apps. It delegates to the `cf-auth init` flow. Scaffolded apps include an `AGENTS.md` runbook, and production setup is one command: `npx --package @cf-auth/cli@latest cf-auth setup --env production`.

Cloudflare Auth is an independent open-source project and is not affiliated with, endorsed by, or sponsored by Cloudflare.
